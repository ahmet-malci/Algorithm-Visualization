# 🚀 Algorithm Visualization Platform

An interactive, web-based educational tool designed to visualize the step-by-step execution of fundamental graph and data compression algorithms. Built entirely with Vanilla JavaScript and HTML5 Canvas, this platform brings complex computer science concepts to life through dynamic animations.

## 📌 Modules & Algorithms

### 1. Graph Algorithms
Visualize pathfinding and Minimum Spanning Tree (MST) algorithms on custom or randomly generated graphs.
* **Dijkstra's Algorithm:** Shortest path finding (Greedy approach).
* **Bellman-Ford Algorithm:** Shortest path with negative weight cycle detection.
* **Prim's Algorithm:** Minimum Spanning Tree generation.
* **Kruskal's Algorithm:** Minimum Spanning Tree using disjoint sets.
* *Highlights:* Directed/Undirected graph toggle, negative edge weights support, real-time distance updates.

### 2. Compression Algorithms
Visualize how text is encoded and decoded using lossless data compression techniques.
* **Huffman Coding:** Frequency-based binary tree generation and bit-stream encoding/decoding.
* **LZW (Lempel-Ziv-Welch):** Dictionary-based compression and decompression step-by-step tracking.
* *Highlights:* Dynamic binary tree drawing, real-time dictionary table updates, live bit-stream tracking.

## ✨ Key Features
* **HTML5 Canvas Rendering:** Smooth, dynamic drawing of nodes, edges, and trees without relying on external visualization libraries.
* **Step-by-Step Execution:** Manually trace algorithms step-by-step to thoroughly understand the underlying logic.
* **Auto-Play Mode:** Sit back and watch the algorithm traverse data structures automatically.
* **Detailed Event Logs:** A dedicated console log panel tracking mathematical state changes (distances, dictionary additions) in real-time.

## 🛠️ Technologies Used
* **Frontend:** HTML5 (Canvas API), CSS3, Vanilla JavaScript (ES6+)
* **Architecture:** Zero-dependency, pure client-side application.

## 🚀 How to Run
This project is completely client-side. No backend server or package installation is required.
1. Clone this repository.
2. Navigate to either the Graph or Compression folder.
3. Open `index.html` in any modern web browser to start exploring.
